# 🧪 Liquor Flavor Chemistry Index

이 문서는 화학 성분과 그에 따른 관능적 특징(Descriptor), 그리고 농도에 따른 뉘앙스 차이를 정의합니다.

## 1. Esters (에스테르 - 과일향 계열)
| 성분명 (Chemical Name) | 주요 향미 (Descriptor) | 역치 (Threshold) | 농도별 뉘앙스 (Nuance by Concentration) |
| :--- | :--- | :--- | :--- |
| **Ethyl Acetate** | 솔벤트, 파인애플, 배 | 30 ppm | 저농도: 은은한 배, 과일 향 / 고농도: 접착제, 매니큐어 리무버, 자극적인 향 |
| **Isoamyl Acetate** | 바나나, 풍선껌 | 1.6 ppm | 저농도: 서양배, 달콤한 과일 / 고농도: 인공적인 바나나 우유, 강한 풍선껌 |
| **Ethyl Hexanoate** | 붉은 사과, 아니스 | 0.2 ppm | 신선한 사과 껍질 향, 약간의 허브 느낌 |
| **Ethyl Octanoate** | 꿀, 왁스, 살구 | 0.9 ppm | 숙성된 과일의 단내, 꼬냑 같은 중후함 |

| **Propanol** | 거친 알코올, 흙내 | - | 날카로운 알코올 취, 숙성 부족 시 두드러짐 |
