# CJ Young Sommelier - Knowledge Base

This directory is reserved for reference materials, documentation, and knowledge resources related to liquor analytics.

## Suggested Contents

- **Chemical References**: Documentation about chemical compounds and their impact on flavor
- **Sensory Standards**: Industry standards for sensory evaluation
- **Training Materials**: Guides for tasters and analysts
- **Product Documentation**: Detailed product specifications and history
- **Research Papers**: Relevant scientific papers on liquor chemistry and sensory science

## Usage

Store PDF, text, or markdown files here that can serve as reference materials for:
- Understanding chemical-sensory relationships
- Training machine learning models
- Generating more accurate AI reports
- Quality control benchmarks
